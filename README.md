# Earning Spain — GitHub Pages Starter

This folder is a **ready-to-publish** website for GitHub Pages.

## What you get
- Sticky top navigation
- Scroll progress bar
- Photo-first sections for Tarragona, Barcelona, Madrid, Toledo
- Two bottom photo boxes per city
- Mobile responsive layout

## 1) Publish (fastest, no terminal)
1. Create a new GitHub repository (example name: `earning-spain`).
2. Upload the contents of this folder to the repo root.
3. Enable GitHub Pages:
   - Repo → **Settings** → **Pages**
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
4. Wait for the green success message and click the URL.

## 2) Add your photos
Put your images into: `assets/images/`

Then edit: `assets/css/styles.css` under **IMG-SETUP** to match your filenames.

Suggested filenames:
- hero.jpg
- tarragona1.jpg, tarragona2.jpg, tarragona_b1.jpg, tarragona_b2.jpg
- barcelona1.jpg, barcelona2.jpg, barcelona_b1.jpg, barcelona_b2.jpg
- madrid1.jpg, madrid2.jpg, madrid_b1.jpg, madrid_b2.jpg
- toledo1.jpg, toledo2.jpg, toledo_b1.jpg, toledo_b2.jpg

## 3) Edit text
Edit `index.html` and change the paragraphs to whatever you want.
